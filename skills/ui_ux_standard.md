# Skill: Enterprise UI/UX Design System Standards

## Description

Applies professional, high-end visual patterns and accessibility parameters to client-side interface files.

## Guidelines

- **Visual Grid:** Enforce structural layout configurations, consistent padding/margins, and fluid typography.
- **Interactive States:** Require immediate visual feedback for all interactive elements (active, hover, disabled, loading).
- **Responsive Architecture:** Enforce strict mobile-first or highly responsive utility classes across all breakpoints.
