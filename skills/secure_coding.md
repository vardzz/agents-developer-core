# Skill: Advanced Defensive Secure Coding Practices

## Description

Injects deep programmatic safeguards directly into application source code logic to prevent unauthorized intrusion.

## Guidelines

- **Input Sanitization:** Validate all incoming network patterns against strict, strongly-typed object schemas before execution.
- **Cryptography:** Ensure all password handling, token strategies, and sensitive session management use industry-standard hashing and encryption patterns.
- **SQLi/XSS Block:** Ensure all data operations utilize parameterized configurations or strict object-relational abstraction layers to eliminate injection surfaces.
