# Skill: High-Throughput Performance Optimization

## Description

Analyzes execution logic to ensure microsecond efficiency, structural scaling, and minimal footprint.

## Guidelines

- **Memory & Caching:** Minimize repeated data execution blocks, maximize memory reuse, and employ cache strategies where appropriate.
- **Query Stripping:** Optimize database interactions to retrieve only required records and fields without N+1 query overhead.
- **Asset Overhead:** Enforce code splitting, lazy loading paths, and minimized execution layers to maximize speed.
